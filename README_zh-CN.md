# PIXAL 2.0

PIXAL 2.0 — 由정성윤创建的AI代理

使用 HuggingFace `google/gemma-4-E2B-it-assistant` 模型的多模态AI代理。

## 功能

- 🧠 思考级别控制：低/中/高/最大下拉菜单
- 🎵 多模态：图片、视频、音频文件与文本一起提交（Python处理）
- 🔧 14种工具：代码执行、Shell、文件操作、图像/音频/视频处理等
- 📊 手风琴式思考过程显示
- 📡 流式输出
- 🐍 Python沙箱：所有包已安装，可写文件系统

## 部署

部署到 Tencent Cloud EdgeOne Makers：
1. 访问 console.cloud.tencent.com/edgeone
2. 创建新项目并关联此仓库
3. 设置环境变量 `HUGGING_FACE_HUB_TOKEN`
